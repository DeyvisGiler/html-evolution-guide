Introducción al HTML

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear páginas web y describe la estructura básica mediante el uso de etiquetas.

Desde su creación en 1991 por Tim Berners-Lee, HTML ha evolucionado significativamente, adaptándose a las necesidades cambiantes de la web y sus usuarios.

¿Sabías que? La primera versión de HTML tenía solo alrededor de 20 etiquetas.
