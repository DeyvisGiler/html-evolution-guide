Todas las versiones del HTML

HTML 1.0 1991

Primera Versión

Fundacional

La primera versión de HTML, creada por Tim Berners-Lee, era muy básica y solo incluía alrededor de 20 etiquetas para estructurar texto, enlaces e imágenes.

20 etiquetas Texto básico
Tim Berners-Lee

HTML 2.0 1995

Primer Estándar

Interactividad

Esta versión introdujo formularios y tablas, permitiendo una mayor interactividad y organización del contenido en las páginas web.

Formularios Tablas Interactividad
RFC 1866

HTML 3.2 1997

Dinamismo Web

W3C Recomendación

HTML 3.2 añadió soporte para scripts y applets, lo que permitió a los desarrolladores crear páginas web más dinámicas y funcionales.

Scripts Applets Dinámico
Primer W3C

HTML 4.01 1999

Revolución Web

Separación CSS

Esta versión mejoró la accesibilidad y separó el contenido de la presentación, promoviendo el uso de hojas de estilo CSS para el diseño visual.

CSS Accesibilidad Separación
CSS Integration

XHTML 1.0 2000

Era XML

Sintaxis Estricta

XHTML reformuló HTML como una aplicación XML, lo que permitió una mayor estricticidad en la sintaxis y facilitó la interoperabilidad con otros sistemas basados en XML.

XML Sintaxis estricta Interoperabilidad
Aplicación XML

HTML5 2014

Web Moderna

Estándar Actual

HTML5 introdujo nuevas etiquetas semánticas, multimedia y APIs para mejorar la interactividad y funcionalidad web, convirtiéndose en el estándar actual.

Semántica Multimedia APIs
