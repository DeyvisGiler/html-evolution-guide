Historia del HTML

Descubre el origen y desarrollo del lenguaje que revolucionó la web mundial.

Creación: 1991

Creador: Tim Berners-Lee

Tim Berners-Lee

HTML fue creado en el año de 1991 por Tim Berners-Lee en la Organización Europea para la Investigación Nuclear (CERN). Su objetivo era facilitar el intercambio de documentos científicos a través de una red global, lo que llevó al desarrollo del World Wide Web.

Su primera versión, el HTML 1.0, era muy básica y solo incluía alrededor de 20 etiquetas para estructurar texto, enlaces e imágenes.

Lugar

CERN, Ginebra, Suiza

Año clave

1991 - Primera propuesta de HTML

Evolución del HTML a lo largo del tiempo

Desde su creación, HTML ha pasado por varias versiones que han ido mejorando sus capacidades y adaptándose a las necesidades de la web.

HTML 1.0 1991

Fundacional

Primera versión con alrededor de 20 etiquetas básicas para estructurar texto, enlaces e imágenes.

20 etiquetas Básico

HTML 2.0 1995

Estándar RFC

Introdujo formularios y tablas, permitiendo una mayor interactividad y organización del contenido.

Formularios Tablas

HTML 3.2 1997

W3C

Añadió soporte para scripts y applets, lo que permitió crear páginas web más dinámicas y funcionales.

Scripts Applets

HTML 4.01 1999

Revolución

Mejoró la accesibilidad y separó el contenido de la presentación, promoviendo el uso de CSS para el diseño visual.

CSS Accesibilidad

XHTML 1.0 2000

XML

Reformuló HTML como una aplicación XML, permitiendo mayor estricticidad en la sintaxis y facilitando la interoperabilidad.

XML Sintaxis estricta

HTML5 2014

Actual

Introdujo nuevas etiquetas semánticas, multimedia y APIs para mejorar la interactividad y funcionalidad web, convirtiéndose en el estándar actual.

Semántica Multimedia APIs

Impacto mundial

HTML ha sido fundamental para el desarrollo de la World Wide Web, permitiendo:

  • Creación de páginas web
  • Documentación científica en línea
  • Comercio electrónico
  • Redes sociales
  • Aplicaciones web complejas
